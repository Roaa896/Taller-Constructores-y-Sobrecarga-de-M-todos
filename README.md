*INTRODUCCIÓN:*        
En este taller se abordan conceptos fundamentales de la Programación Orientada a Objetos en Java, específicamente el uso de constructores y la sobrecarga de métodos. Los constructores permiten inicializar objetos de una clase de manera adecuada, mientras que la sobrecarga facilita la creación de múltiples métodos o constructores con el mismo nombre, pero con diferentes parámetros, brindando mayor flexibilidad al programa. El objetivo de estos ejercicios es comprender cómo crear e inicializar objetos, así como aplicar distintas formas de interacción mediante la sobrecarga. Esto contribuye al desarrollo de habilidades en el diseño de programas más organizados, reutilizables y eficientes, sirviendo como base para el manejo avanzado de clases y objetos en Java.
